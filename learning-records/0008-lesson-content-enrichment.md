# レッスン6〜17の内容拡充

「下書きなので無視して再構築してよい」との指示で、初期フェーズ2〜4で薄く作られていたレッスン6〜17（工具・施工・検査・法令・計算・技能）を、レッスン1〜5と同じ密度（会話劇・scene-card・trivia-box・reveal暗記・クイズ2〜3問）まで再構築した。

拡充と同時に、Web検索で数値の裏取りを行い、以下の追加の事実誤りを発見・修正した（レビュー後も残っていた誤り）：

- **D種接地工事**：「単独100Ω／共同接地2Ω」という誤った記述 → 正しくは「原則100Ω以下、動作時間0.5秒以内の漏電遮断器があれば500Ω以下に緩和」（`lessons/0009`, `reference/numbers-to-memorize.html`）
- **漏えい電流の代替基準**：「1mA/kW」という誤記 → 正しくは電圧区分に関わらず一律「1mA以下」（`lessons/0010`, `reference/numbers-to-memorize.html`）

新たに検証済みの数値として追加した内容：
- リングスリーブのサイズ・圧着マーク対応表（1.6mm×2本のみ「○」の例外）
- 分岐回路の種類（15A/20A配線用遮断器/20Aヒューズ/30A/40A/50A）と電線太さ・コンセント定格の対応
- 電圧降下の近似式係数（単相2線35.6・単相3線17.8・三相3線30.8）とその導出理由
- 漏電遮断器の感度電流・動作時間（人体保護用は30mA以下・0.1秒以内）
- 認定電気工事従事者制度（第2種取得後3年実務または認定講習で取得、自家用電気工作物600V以下の簡易電気工事が可能に）
- 技能試験の欠陥判断基準12区分（学習用要約、公式PDFへのリンク併記）

CSS面では `.formula-box` と `.symbol-box` を各レッスンのインラインstyleから `assets/scene-story.css` に集約し、DRY原則に沿って重複を除去した。

**Implications:** 全17レッスンが同水準の密度になった。数値系の記述は今回で検証済みだが、年度更新（技能候補13問・法令の細目）は毎年1月頃に公式資料と照合が必要。
